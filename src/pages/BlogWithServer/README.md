# Blog with server

## A beautiful prototype of a blog for posting articles with Markdown support and simple backend

> #### what is implemented:
>
> - Create/delete post 
> - Automatic cover addition based on category on backend
> - Input data checking
> - Custom Use context hook to transfer data
> - Boostarp form 
> - Markdown converter
